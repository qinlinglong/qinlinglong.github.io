用hexo如何在github上搭建博客?

发表于 2016-08-12 | 分类于 nodejs

hHgitub提供了一个静态页面的功能,可以搭建属于自己静态博客,基于nodejs的hexo刚好提供了一个简单可依赖的博客框架,刚好在gitub上搭建一个属于自己的主页.

前期准备

  1. github账号
  2. 安装hexo脚手架

步骤

创建项目

在github上创建一个新的仓库，名字格式为 XXXX.github.io 。XXX必须是你的用户名。

安装hexo-cli

初始化项目

  1. git init
  2. git remote add origin https://github.com/XXXX/XXXX.github.io.git(不执行)
  3. hexo init

注意：key和value之间必须有一个空格

  2. hexo generate

  3. hexo deploy（这个过程很慢，请耐心等待）
